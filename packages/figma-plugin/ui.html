▶ How it works
  1. Your AI client launches the MCP server (npx -y figma-relai) — and that server hosts the relay on port 9055. Nothing else to install or keep running.
  2. This plugin simply dials ws://localhost:9055. Either side can start first; both reconnect automatically.
  3. Talk to your AI — it discovers this plugin's room and pairs on its own. Nothing to copy.
  4. Contributors: clone the repo and run bun setup to point your MCP config at a local build.
